Swagger

The swagger (Open API) definition of the interface specification can be found here, as well as below.

Many tools are available for processing swagger files, and using a code generation tool to create a boilerplate client or server can be an excellent starting point for development. We highly recommend looking at swagger-codegen for a generator in your preferred language when starting out.

Swagger definition

swagger: "2.0"
info:
  description: "The Electrum CRM Portal Interface describes a server interface against\
    \ which calls can be
                  made to retrieve information from a backend\
    \ CRM system. The intended use cases include the lookup of
                 \
    \ customer-linked account or purchase history data from a bank or retailer's CRM\
    \ environment. It is not
                  intended that the CRM system is hosted\
    \ by Electrum, but rather that this interface abstracts the logic
          \
    \        required to execute a set of predefined CRM queries, hence simplifying\
